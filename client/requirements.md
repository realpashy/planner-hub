## Packages
date-fns | Date manipulation and formatting
framer-motion | Page transitions, layout animations, and micro-interactions
canvas-confetti | Celebration animation when all tasks are completed
@types/canvas-confetti | TypeScript definitions for canvas-confetti
uuid | Generating unique IDs for offline storage
@types/uuid | TypeScript definitions for uuid
vaul | Beautiful mobile-first bottom drawers

## Notes
- The application is entirely offline-first, using localStorage for data persistence.
- strict RTL layout with Arabic language as default.
- Uses custom date formatting to guarantee Latin digits (e.g., 2026 instead of ٢٠٢٦).
