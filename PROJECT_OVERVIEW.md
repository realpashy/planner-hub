# Project Overview

## Purpose
Planner Hub is a modular RTL planning platform for Arabic/Hebrew users focused on a premium, calm, mobile-first planning experience. The product is designed to grow into a SaaS ecosystem with interconnected modules and automation suggestions.

## Main Modules
- المخطط الأسبوعي (active)
- الميزانيّة الشهرية (active, expanding)
- العادات (planned standalone)
- منظم الحياة (planned)
- التخطيط الشهري (planned)
- أهداف السنة (planned)
- تتبع المهام (planned)
- مخطط السفر (planned)
- وجبات الأسبوع (planned)

## UX Philosophy
- RTL-first design
- No English user-facing copy
- Latin digits in all numbers and time
- Minimal friction and clear hierarchy
- Rewarding micro-interactions without noise
- Mobile-first, desktop-strong

## Current Core Features
- Auth-protected dashboard with active/planned modules
- Weekly planner with selected-day model, tasks, events, focus tags, notes, habits, summaries, charts, and template popup flow
- Budget planner with transaction lifecycle, recurring logic, category settings, savings goals/contributions, interactive overview chart, and financial insights
- Global stacked notifications for user actions across modules
- Light/dark themes with enhanced light-mode contrast and calmer visual balance
- Local persistence with cloud sync for authenticated users

## Product Direction
- Keep modules loosely coupled but link-ready
- Move toward shared cross-module entities
- Expand auth/subscription capabilities over time
- Keep automations suggestion-based (accept/dismiss), never silent task creation
