# Project Overview

## Purpose
Planner Hub is a modular RTL planning platform for Arabic/Hebrew users. The goal is a premium, calm, mobile-first planning experience that can evolve into a SaaS product with interconnected modules and automation-assisted suggestions.

## Main Modules
- المخطط الأسبوعي (Weekly Planner) - active
- الميزانية (Financial Planner) - active and expanding
- العادات (Habit Module) - planned as standalone
- منظم الحياة - planned
- التخطيط الشهري - planned
- أهداف السنة - planned
- تتبع المهام - planned
- مخطط السفر - planned
- وجبات الأسبوع - planned

## UX Philosophy
- RTL-first interface
- No English UI copy for end users
- Latin digits only in UI numbers and times
- Minimal, clear, motivating layout
- Subtle motion and rewarding feedback
- Mobile-first without sacrificing desktop clarity

## Core Features (Current)
- Dashboard with modules grid
- Weekly planner with day strip, tasks, events, focus tags, notes, habits, summary widgets, charts
- Planner templates flow shown only when entering weekly planner setup
- Financial planner with focused tabs, powerful monthly operations management, category customization, savings goals, Arabic month context, and smart warnings
- Local persistence (localStorage)
- Theme support and responsive design

## Product Direction
- Transition from local-only data to shared entities across modules
- Add auth, subscriptions, and persistence later
- Keep all module interactions suggestion-based (no silent destructive automation)
