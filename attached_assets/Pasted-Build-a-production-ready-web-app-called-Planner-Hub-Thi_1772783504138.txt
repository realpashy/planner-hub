Build a production-ready web app called "Planner Hub".

This is a premium RTL planner web app for Arabic and Hebrew users.

IMPORTANT RULES

Do NOT build authentication.
Do NOT build subscriptions.
Do NOT build paywalls.

The architecture must still allow those features later.

For now the app runs fully offline using local persistence.

Use a clean modern stack:

React
TypeScript
Vite
TailwindCSS
Lucide icons
localStorage persistence

The code must be modular and scalable.

--------------------------------------------------

APP PURPOSE

This is NOT an admin dashboard.

This must feel like a premium lifestyle planner app.

It should feel:

• motivating
• elegant
• modern
• calm but productive
• mobile-first
• emotionally rewarding

--------------------------------------------------

LANGUAGE

Arabic and Hebrew only.

Both are RTL.

No English UI.

Add language switcher.

Persist language locally.

Use fonts:

Arabic: Tajawal
Hebrew: Heebo

Numbers must ALWAYS use Latin digits.

Example:

2026
